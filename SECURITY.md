# Security Policy

This repository is intended to be public for website demo purposes.

## Rules

- Do not commit secrets, credentials, private keys, or tokens.
- Do not commit internal customer data or exports.
- Keep only public web assets and redesign code in this repository.

## If Sensitive Data Is Committed

1. Remove exposed data from the repository immediately.
2. Rotate impacted credentials immediately.
3. Rewrite git history if needed.
4. Re-verify GitHub secret scanning alerts are clear.
